macOS

macOS — це операційна система, призначена для лінійки комп'ютерів Mac. Вона базується на ядрі Darwin, що успадковане від технологій NeXT STEP, та використовує фірмове графічне середовище Aqua. Головними перевагами macOS вважаються високий рівень безпеки, стабільність роботи, елегантний дизайн інтерфейсу і, що критично важливо для користувачів Apple, безшовна інтеграція з іншими продуктами компанії через функції на кшталт Handoff та Universal Clipboard.

iOS / iPadOS

Для мобільних пристроїв розроблено дві основні системи.iOS є операційною системою для смартфонів iPhone і служить фундаментом для всієї мобільної платформи Apple. Вона відома своєю інтуїтивністю, високою продуктивністю та суворим контролем над магазином додатків. iPadOS , своєю чергою, є адаптованою версією iOS для планшетів iPad. Вона включає додаткові можливості, спеціально розроблені для великого екрана та форм-фактора планшета, такі як покращена багатозадачність (Split View та Slide Over), настроюваний Dock, глибша робота з файлами та підтримка аксесуарів, включаючи Apple Pencil та клавіатури.

watchOS

watchOS — це спеціалізована операційна система для пристроїв, що носяться, Apple Watch. Вона зосереджена на функціях моніторингу здоров'я, відстеження фізичної активності та швидкої інтеграції з iPhone для сповіщень і зв'язку, пропонуючи унікальний досвід користувача, оптимізований для маленького екрана на зап'ясті.
